# install

1. clone the project

2. cd into the project folder

3. copy .env.example to .env

4. config database and email in .env file

5. run `composer install`

6. run `php artisan migrate:fresh --seed` (if it failed re run it)

7. run `php artisan serve`

8. enjoy
